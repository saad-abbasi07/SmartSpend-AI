# SmartSpend-AI

SmartSpend-AI is a machine learning-powered personal finance tool that predicts monthly spending, analyzes transaction patterns, and provides smart budgeting recommendations for users.

---

## Features

- Predict monthly spending per user using ML models
- Analyze transaction patterns and category-wise expenses
- Smart budget recommendations based on historical data
- REST API for web-based app integration
- Easily extendable with additional features like recurring transactions and location-based analysis

---

## Dataset

- Transaction-level data including amount, category, date, user, and merchant
- Aggregated monthly spend per user for model training
- Synthetic or anonymized datasets recommended for privacy
